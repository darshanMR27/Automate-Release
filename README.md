# Automate Release Branching:

## Prerequisites

* Java JDK 11 version
* Apache Maven
* IDE IntelliJ or Eclipse (Optional)

## Update your Jira Credentials in application yaml file

:man: auth.username=

:man: auth.password=

### Build applciation:

:smile: mvn clean install

### Run Application:

:smile: mvn spring-boot:run
